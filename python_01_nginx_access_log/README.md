## Perplexity. Python: Task-01.

Напишите скрипт, который читает файл `access.log` (стандартный **Nginx** лог), подсчитывает количество запросов по 
статус-кодам (200, 404, 500 и т.д.) и выводит топ-3 статусов с количеством. Используй `collections.Counter` и `argparse`
для пути к файлу (по умолчанию `./access.log`). 
Пример лога:

```txt
127.0.0.1 - - [01/Feb/2026:21:00:00 +0300] "GET / HTTP/1.1" 200 1234 "-" "curl/7.68"
```
Ожидаемый вывод:

```txt
200: 150
404: 20
500: 5
```
